## Images hosting
1. Keep the image name in the user's attributes
2. Save the file into the web server
3. Rebuild the complete path using basic path (to store in the config file) + file name

#### Example : 
*In application.yml*\
filePath: "/var/www/tp"\
fileUrl: "http://127.0.0.1:8081/tp"
